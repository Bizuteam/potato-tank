# potato-tank
Projet de Shoot'Em Up en C avec Allegro

### Installer Allegro (merci [Eresia](https://www.github.com/Eresia "Github de Eresia"))
```
git clone https://github.com/liballeg/allegro5.git
cd allegro
git checkout 5.0
mkdir build
cd build
cmake ..
make
sudo make install
```

### Compiler et lancer le jeu
```
make
./al.out
```
